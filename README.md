# Advanced Student Calculator

A responsive calculator web application created while learning Git and GitHub.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage Calculation
- Power Calculation
- Square Root
- Dark Mode Toggle
- Keyboard Shortcuts
- Responsive Mobile Design

## Technologies Used

- HTML
- CSS
- JavaScript

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Enter | Add Numbers |
| Escape | Clear Fields |

## How To Run

1. Download or clone the repository
2. Open `index.html` in browser
